Replicaset'in ne olduğu, nasıl bir mimariye sahip olduğu, ne amaçla kullanıldığına dair detayları öğrenmek için [Learn @ Enterprisecoding](http://learn.enterprisecoding.com/) web sitemizi ziyaret edebilirsiniz.
